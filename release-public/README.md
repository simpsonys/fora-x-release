# FORA-X Public Release

This repository contains the public releases and documentation for FORA-X, a clean-room, Windows-first, keyboard-first, dual-pane file explorer.

## Downloads

You can find the latest release assets (portable ZIP, MSI installer) on the [Releases page](https://github.com/simpsonys/fora-x-release/releases).

## Documentation

Selected public documentation is available in this repository.

## Source Code and Contributions

The source code for FORA-X is maintained in a private repository. This public repository is for distribution only.
